DaChat Widget – Manual Test Page

This page embeds the live DaChat widget. Click the chat bubble in the bottom-right to start a conversation.

Test checklist:
  1. Open the widget — a new contact session is created automatically.
  2. Type a message and send it. Open the operator dashboard in another tab and verify it arrives.
  3. From the dashboard, reply — the widget should receive it in real time.
  4. Start typing in the widget input — the dashboard should show "Contact is typing…" after ~0.5 s.
  5. Have the dashboard operator type back — widget should show "Operator is typing…".
  6. Send a message from the dashboard; when it is visible in the widget the receipt:read event fires (double-tick appears on the dashboard message if the operator reads the contact's message).
  7. Click the 📎 paperclip button, choose a small image or file, and verify a file bubble appears in the thread.

The server must be running at http://localhost:3001 and this file should be served via npm run dev inside packages/widget (port 3002), which proxies all API and WebSocket requests to the server automatically.
